# mooc复制粘贴助手

【功能一 刷题】删除多余广告、复制全部题目、复制题干或选项、搜索题干或选项：  
① 删除多余广告：DEL删除多余元素；   
② 复制全部题目：COPY一键复制所有题目和选项的文本至剪切板中，暂不支持图片复制。(不记录正确答案)  ；  
③ 复制题干或选项：**鼠标移至题干或选项上方即可显示“复制”、“必应搜索”两个按钮。**    
注：DISABLE可以禁用“复制”、“必应搜索”避免影响做题，禁用后将变成ABLE按钮。    


【功能二 视频】任意倍速：  
① +0.25倍速：按w键；  
② -0.25倍速：按s键；  
③ 切换整数倍速：按1/2/3/4键。
